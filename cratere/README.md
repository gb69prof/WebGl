# Museo 3D – Cratere Greco (WebGL)

## Se resta fermo a 0% su iPad
Quasi sempre stai aprendo la pagina come `file://` oppure non sei in HTTPS.
Su iPad Safari il caricamento del GLB spesso non funziona da locale: usa **GitHub Pages** (HTTPS).

## GitHub Pages
1. Crea un repo
2. Carica `index.html` e la cartella `assets/`
3. Settings → Pages → Deploy from a branch → `main` (root)

## Comandi
- **Osserva**: 1 dito ruota, 2 dita zoom/pan
- **Passeggia**: joystick sinistro muove, destro guarda
- **Reset**: vista iniziale
