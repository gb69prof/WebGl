# Museo 3D – Cratere Greco (WebGL)

Questa versione NON usa CDN: include `three` e i loader in locale (cartella `libs/`).
È più stabile su GitHub Pages e su iPad.

## Struttura
- index.html
- assets/cratere-greco4.glb
- libs/three.module.min.js
- libs/OrbitControls.js
- libs/GLTFLoader.js

## Pubblicazione GitHub Pages
Settings → Pages → Deploy from a branch → main / (root) oppure /docs (se lo preferisci).
