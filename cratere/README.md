# Museo 3D – Cratere Greco (WebGL)

## Come pubblicare su GitHub Pages
1. Crea un nuovo repository su GitHub (es. `museo-cratere`).
2. Carica **tutto** il contenuto di questa cartella nel repo:
   - `index.html`
   - `assets/cratere-greco4.glb`
3. Vai su **Settings → Pages**:
   - Source: `Deploy from a branch`
   - Branch: `main` (root)
4. Apri l'URL di GitHub Pages che ti verrà indicato.

## Comandi (iPad)
- **Osserva**: un dito ruota, due dita zoom/pan.
- **Passeggia**: joystick sinistro muove, joystick destro guarda.
